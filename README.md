Step into a world of discovery. The Country Catalogue is a colourful, beginner‑friendly website where you can explore continents, learn fun facts, and jump between pages as if you're travelling the globe. Whether you're curious about Africa, Asia, Europe, or anywhere else, this site gives you a simple and enjoyable way to explore the world.

This project was created as a learning journey in HTML, CSS, and building multi‑page websites — and now it’s fully live on the web for anyone to visit.


## 🌐 Visit the Website
Come explore the world here:
👉 **https://pikachumaster16.github.io/The-Country-Catalogue/**
The site opens on the main homepage, where you can choose a continent and start your adventure.


## ✨ What You’ll Find on the Website
- A friendly homepage that welcomes you in  
- Separate pages for each continent  
- Custom colours and styles for every region  
- Easy navigation so you can jump around the world  
- A simple, clean design that’s perfect for learning and exploring

Every continent page has its own CSS file, giving each section its own personality and look.


## 📁 What’s Inside the Project
This website is made up of simple HTML and CSS files:

- `index.html` — loads the site  
- `Home_Page.html` — the main landing page  
- The continent html files — (`Africa.html`, `Asia.html`, `Europe.html`, `North_America.html`, `South_America.html`, `Oceania.html`)  
- Matching CSS files for each continent  
- `styles.css` — global styling  
- `Icon.png` — the site’s icon  

Everything works together to create a smooth, colourful experience.


## 🎯 Why This Project Exists

The Country Catalogue was created to make learning about the world fun, simple, and interactive. Instead of reading long paragraphs or scrolling through walls of text, this website lets visitors explore the globe one continent at a time. Each page has its own style and personality, making the experience feel like a small adventure.

This project also represents a personal milestone — a chance to practice building real websites, organizing multiple pages, designing with CSS, and publishing something fully online for others to enjoy. It’s a mix of creativity, curiosity, and learning all in one place.
 
It’s a simple project, but it shows how much you can create with just HTML and CSS.


## 🚀 Try It Yourself
